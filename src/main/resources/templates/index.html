Pizza varieties generator

 List of Available Pizzas
 Get varieties of pizzas based on maximum price
REST interface
  • Get List of Pizzas: GET /pizzas
  • Update list of pizzas PUT /pizzas
    cURL example:
    curl --location --request PUT 'localhost:8080/pizzas' \
--header 'Content-Type: application/json' \
--data-raw '[
