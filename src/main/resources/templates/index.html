Pizza varieties generator

 List of Available Pizzas
 Get varieties of pizzas based on maximum price
REST interface
  • Get List of Pizzas: GET http://localhost:8080/pizzas(when running on localhost:8080)
  • Update list of pizzas PUT http://localhost:8080/pizzas(when running on localhost:8080)
    cURL example when running on localhost:8080 (linux):
    curl --silent --request PUT 'localhost:8080/pizzas' \
--header 'Content-Type: application/json' \
--data-raw '[
    {
        "name": "Szupr\u00E9m - nagy m\u00E9ret",
        "price": 4490
    },
    {
        "name": "Texas - nagy m\u00E9ret",
        "price": 4490
    },
    {
        "name": "Margarita - nagy m\u00E9ret",
        "price": 3790
    }
]'
    cURL example when running on localhost:8080 (Windows):
    curl --silent --location --request PUT "localhost:8080/pizzas" ^
--header "Content-Type: application/json" ^
--data-raw "[{\"name\": \"Szupr\u00E9m - nagy m\u00E9ret\",\"price\": 4490},{\"name\": \"Texas - nagy m\u00E9ret\",\"price\": 4490},{\"name\": \"Margarita - nagy m\u00E9ret\",\"price\": 3790}]"
